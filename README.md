fibjs更少回调，传说比那XXJS快8快，只是传说听到!!
本项目是基于fibjs开发的Blog;

### 1.前端

- 模板渲染(juicer)

- markdown(正则比较渣，自己写的)

- 通讯(ajax)

### 2.后端

- 数据库(sqlite/mysql)

- 简易的MVC结构(分离了router/modules/dbs);


### 启动服务:
```
osx: sudo js server.js
windows: fibjs.exe server.js
```

### 初始化数据库:
```
osx: js init.js
windows: fibjs.exe init.js
```
